6.3 最大鉄筋量【コンクリート標準 11.4.2】

曲げモーメントの影響が支配的な部材は、引張鉄筋の降伏よりもコンクリートの

圧縮破壊が専攻するような脆性的な破壊を生じない引張鉄筋とするために、引張鉄

筋比を釣合鉄筋比の75%以下とする。

ただし、本計算例では地震時の照査において、破壊形態に応じた部材性能(非線

形)を評価しているため、上記条件を検討する必要は無いが、以下にその検討方法

を示す。

(1) 釣合鉄筋比pbの算定

pb = ((α ε'cu - N'd ) f'cd + Asc ・ σ's ) × 100
ε'cu + εsy b・d・f'cd fsyd b・d fsyd

ここに、α = 0.88 - 0.004f'ck ただし、α≦0.68

ε'cu : コンクリートの終局ひずみ

= 115 - f'ck ≦ 0.0035
30000

εsy : 引張鉄筋の降伏ひずみ

= fsyd / Es

N'd : 設計軸方向圧縮力

（本計算例は、N=0の場合およびNmaxを用いて算定する）

b : 部材の幅

d : 部材の有効高さ

f'cd : コンクリートの設計圧縮強度

= f'ck / γc

fsyd : 鉄筋の設計引張降伏強度

Es : 鉄筋のヤング率

Asc : 圧縮鉄筋量

σ's : 圧縮鉄筋の応力

≦fsyd

・安全係数及び修正係数

材料係数 γc = 1.3

γs = 1.0
