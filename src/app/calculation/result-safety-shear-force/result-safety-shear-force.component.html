(2)せん断力に対する照査 {{title}}【コンクリート標準　7.2.3】

せん断力を受ける部材の照査は、設計せん断力Vdを設計応答値とし、部材毎にa/dの

値により、設計限界値を設計せん断耐力Vyd腹部コンクリートの設計斜め圧縮破壊

Vwcdまたは、設計せん断圧縮破壊耐力Vddとして、次式により行う。

1)a/d≧2.0の場合

(a)設計せん断による照査

γi・Vd/Vyd≦1.0

ここに、γi：　構造物係数

Vd：設計せん断力

Vyd：設計せん断耐力

Vyd = Vcd + Vsd

Vyd = Vcd + Vwd + Vsd

Vcd：せん断補強鋼材を用いない棒部材のせん断耐力

Vsd：せん断補強鋼材により受け持たれる棒部材の設計せん断力

Vwd：せん断補強鋼材により受け持たれる棒部材の設計せん断力

Vsd：鉄骨部分により受け持たれる棒部材の設計せん断耐力

a)せん断補強鋼材を用いない棒部材の設計せん断力

Vcd = βd・βp・βn・fvcd・bw・d / ・γb

ここに、βd：(1000/d)1/4 ただしβd＞1.5となる場合は1.5とする。

βp：(100×Pc)1/3 ただしβp＞1.5となる場合は1.5とする。

βn：1+2M0/Mu(N'd≧0) ただしβn＞2となる場合は2とする。

βn：1+4M0/Mu(N'd＜0) ただしβn＜0となる場合は0とする。

fvcd：0.2(Pcd)1/3

ただし、fvcd＞0.72N/mm2となる場合はfvcd＝0.72N/mm2とする。

bw：腹部の幅

d：有効高さ

Pc：せん断引張鋼材比

As：引張鋼材の断面船

Mu：曲げ耐力

Mo：設計曲げモーメントMdに対する引張縁において、軸方向により発

生する応力を打ち消すのに必要な曲げモーメント。

Nd：設計軸方向圧縮力

γb：部材係数

b)せん断補強鋼材により受け持たれる棒部材の設計せん断耐力

Vsd = Aw・fwyd・d(1.15Ss・γb)

Vwd = Aw・fwyd・d(1.15Ss・γb)

ここに、Aw：区間Ssにおけるスターラップの総断面積

d：有効高さ

Ss：スターラップの配置間隔

γb：部材係数

c)鉄骨部分により受け持たれる棒部材の設計せん断耐力

Vsd = fsvyd・hw・tw / γb

ここに、fsvyd：構造用鋼材の設計せん断降伏強度

hw：鉄骨腹板の高さ

tw：鉄骨腹板の厚さ

γb：部材係数で、1.1とする

(b)腹部コンクリートの斜め圧縮破壊耐力による照査

γi・Vd/Vwcd≦1.0

ここに、γi：構造物係数

Vd：設計せん断力

Vwcd：設計斜め圧縮破壊耐力

Vwcd = fwcd・bw・d / γb

fwcd = 1.25(f'cd)1/2 ≧7.8(N.mm2)

bw：腹部の幅(mm)

d：有効高さ

f'cd：コンクリートの設計圧縮強度(mm)

γb：部材係数

2)a/d＜2.0の場合

γi・Vd/Vdd≦1.0

ここに、γi：構造物係数

Vd：設計せん断力

Vdd：設計せん断圧縮破壊耐力

Vdd = (βd・βn・βw)βp・βa・fdd・bw・d/γb

fdd = 0.19(f'cd)1/2 (Nmm2)

βd = (1000/d)1/4 ≦1.5

βw = 4.2・(100pw)1/3・(a/d - 0.75) / (f'cd)1/2≧0

βn = 1+2Mo/Mu≦2(N'd≧0の場合)

= 1+4M0/Mu≧0(N'd＜0の場合）

βp = ｛1 + (100pc)1/3｝/2≦1.5

βa = 5/｛1 + (a/d)2｝

bw：腹部の幅(mm)

d ：有効高さ

a ：支持部前面から荷重載荷点までの距離(mm)

pc：せん断引張鋼材比 pc = As/(bw・d)

As：引張鋼材の断面積(mm2)

pw：せん断補強鉄筋比 pw = Aw/(bw・Ss)

ただし、pw＜0.002となる場合はpw＝0とする。

Aw：区間Ssにおける部材軸と直行するせん断補強鉄筋の総断面積(mm2)

Ss：部材軸と直行するせん断補強鉄筋の配置間隔(mm)

Mu：曲げ耐力

Mo：設計曲げモーメントMdに対する引張縁において、軸方向力によっ

て発生する応力を打ち消すのに必要なモーメント

N'd：設計軸方向圧縮力

f'cd：コンクリートの設計圧縮強度(N/mm2) (＝f'ck/γc、γc = 1.3)

γb：部材係数

Vydd = Vcdd + Vsdd

ここに、Vydd : せん断スパン比の小さい部材の設計せん断耐力（N）

Vcdd : せん断補強鋼材を用いないせん断スパン比の小さい部材の

設計せん断耐力（N）

Vsdd : せん断スパン比の小さい部材のせん断補強鋼材により受け持た

れる棒部材の設計せん断耐力（N）

a)せん断補強鋼材を用いないせん断スパン比の小さい部材の設計せん断力

Vcdd = 0.65・βa・f'cd1/3・βd・βp・βn・bw・d / γb

ここに、βa = (a / d)-1.166 ただし、βα＞2.244となる場合は2.244とする。

a : 支持部前面から載荷点までの距離（mm）

γb : 部材係数で1.3とする。

( βd, βp, βn, f'cd, bw, d は上を参照 )

b)せん断スパン比の小さい部材のせん断補強鋼材により受け持たれる棒部材の

設計せん断力

Vsdd = βs・(Aw・fwyd(sinθs+cosθs)/Ss+Apw・σpw(sinθp+cosθp)/Sp)・z/γb

ここに、βs = 2・(a / d - 0.5) ただし、a/d＜0.5となる場合は0とする。

a/d＜1.0となる場合は1.0とする。

a : 支持部前面から載荷点までの距離（mm）

d : 有効高さ（mm）

γb : 部材係数で1.1とする。

( Aw, Apw, fwyd, σpw, θs, θp, Ss, Sp, z は上を参照 )
