(b)せん断ひび割れの検討【コンクリート標準　10.2.2.5】

1)せん断ひび割れの検討を省略できる部材

永久作用と変動作用が組み合わされた時の設計せん断Vdが、せん断補強鋼材を用い

ない棒部材の設計せん断耐力Vcdの70％よりも小さい場合には、せん断ひび割れの検討

は省略できる。なお、このVcdを求める場合、コンクリートの材料の材料係数γcおよび

γbは1.0とする

2)せん断ひび割れの検討

せん断ひび割れの検討においては、永久作用によるせん断補強筋の応力度σwpdを

表3.4.2に示す限界値σs12以内とすることにより、せん断ひび割れ幅に関する検討を省略

してよい。

σwpd≦σs12

ここに、σwpd： 永久作用によるせん断補強筋の設計応力度(N/mm2)

σwpd = (Vpd + Vrd - kr・Vcd)・s (αr(Vpd + Vrd) - kr・Vcd)・s ・ Vpd + Vcd αr・Vpd + Vcd
Aw・z・(sinθ + cosθ) Vpd + Vrd + Vcd αr・(Vpd + Vrd) + Vcd

σs12： せん断またはねじりひび割れ幅の検討を省略できる部材における永

久作用による鉄筋応力度の制限幅(N/mm2)

Vpd： 永久作用による設計せん断力(N)

Vrd： 変動作用による設計せん断力(N)

Vcd： せん断補強鉄筋を用いない棒部材の設計せん断力(N)

(この場合の部材係数γbは1.0とする。)

kr： 変動作用の頻度の影響を考慮するための係数で0.5とする。

αr = Ar・z
Zs + Ar・z

Zs： 鉄骨の断面係数（mm3）

Ar： 軸方向引張鉄筋の総断面積（mm2）

s: せん断補強鉄筋の配置間隔(mm)

Aw： 区間sにおけるせん断補強鉄筋の総断面積(mm2)

θ： せん断補強鉄筋が部材軸となる角度(＝90°)

z： 圧力応力の合力位置から引張鋼材の図心までの距離でｄ/1.15とする。

d： 有効高さ (mm)
